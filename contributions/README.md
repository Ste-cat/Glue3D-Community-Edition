## What is this?
This is the contributions folder, if you're making any contributions, make a pull request and add an idea via a text file, or a feature via a .sb3 or a .sprite3 file.

> [!WARNING]
> MAKE SURE that the pull window is open when you make a pull request, as if you make a contribution when it's closed you'll have to wait ti'll the next patch or build until it's added
